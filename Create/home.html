The Create Utility

This tool assists the user in generating DDL (Data Definition Language) statements for creating Tables and Databases as per the
user's specifications. Additionally, it offers some DML assistance, as it lets the user alter the table structure in the form of, adding or
removing columns, adding constraints to the table (Primary Key, Candidate Key), by the means of applying the constarint on an existing attribute. It also lets the user drop tables and delete databases.
